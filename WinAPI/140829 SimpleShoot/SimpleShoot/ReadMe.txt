1. 상단에 적군이 왔다 갔다.
2. 하단 플레이어가 키보드 방향(좌우) 이동
3. 스페이스 누르면 플레이어로부터 총알 발사
4. 총알과 적군 충돌 처리
	- 총알은 충돌하면 사라짐
	- 적군은 3방 맞으면 사라짐

<과제>
- 적군이 랜덤하게 위에서 등장해서 아래로 아래로 이동(지나가며 맵밖으로 나가면 삭제)
- 적군과 플레이어가 충돌하면 플레이어 사망

- 총알이 화면 밖으로 나가면 다시쏠수 있다.
- 총알은 100발 (vector를 사용하면 과부하가 걸리나? 그렇다고 하면  array를 이용해본다)
- 충돌되면 총알이 사라진다
- 적군은 3발 맞으면 사라진다
- 적군이 죽으면 게임 종료
